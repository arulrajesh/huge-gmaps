# Changelog

This file will summarize the history of changes to this program.

## [1.1][1.1] - 2017-03-26 - Usage instructions
* Add `outfile` parameter to the `create_map` function so the
  location the final image can be set explicitly.
* Add more detailed usage instructions to `README.md`, and update
  `tests.py` to reflect the new instructions.
* Now scaling images immediately after taking each screenshot
  instead of scaling them all at the end.

## [1.0][1.0] - 2017-03-21 - Alpha
* First release of program. Only tested creating a 25000x25000
  pixel map of Philadelphia from an Ubuntu laptop.


[1.0]: https://github.com/rocheio/huge-gmaps/tree/v1.0
